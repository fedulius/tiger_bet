Текущее UX-ядро MVP

• Single-page кабинет без отдельной навигации

• Sticky-хедер с якорями + кнопка «Обновить сейчас»

• Рекомендации: карточка B, авто+ручное обновление

• Новые рекомендации: бейдж «Новые»

• Избранное: +Добавить → модалка (поиск + чекбоксы)

• Последние прогнозы: карточка A (минимум)

• Empty state: fallback топ-матчи

• Тема: Auto (light/dark)

Следующий шаг: сортировка рекомендаций по умолчанию

A

По времени матча (ближайшие сверху)

Понятно и привычно для пользователя.

B

По уверенности прогноза (высокая сверху)

Сначала самые сильные сигналы.

C

Гибрид: сначала высокая уверенность, затем время

Сильные сигналы, но без потери актуальности по времени.
